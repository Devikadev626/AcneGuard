# AcneGuard

## Objective

AI-powered hospital-based acne classification system for:

- Face Acne
- Neck Acne
- Chest Acne
- Back Acne
- Shoulder Acne

## Severity Levels

- Mild
- Moderate
- Severe
- Very Severe

## User Roles

- Patient
- Doctor
- Technician
- Admin
